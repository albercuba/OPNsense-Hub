Email Settings

Enable either SMTP or Microsoft Graph sendMail. Only one delivery method can be active at a time.

Steps to configure Microsoft Graph sendMail app-only auth

Use this setup only for server-side notification delivery. The app uses Microsoft Graph client credentials, so no user mailbox password is stored and no interactive sign-in is required.

  1. In Microsoft Entra admin center, open App registrations → New registration.
  2. Name the app registration clearly, for example OPNsense Hub Mailer.
  3. Do not add a redirect URI for this mailer app.
  4. Copy the Directory (tenant) ID into Tenant ID.
  5. Copy the Application (client) ID into Client ID.
  6. Create a client secret and paste the secret Value, not the Secret ID.
  7. Add Microsoft Graph `Mail.Send` application permission and grant admin consent.
  8. Enter the sender mailbox address that Graph should send as.
  9. Store and rotate the client secret securely.
